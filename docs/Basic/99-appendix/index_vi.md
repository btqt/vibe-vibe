# Phụ lục

Nơi đây thu thập các tài liệu tra cứu nhanh thường dùng trong quá trình học Vibe Coding. Khi gặp vấn đề, hãy thử tìm câu trả lời ở đây trước.

## Điều hướng phụ lục

| Phụ lục                               | Nội dung                                                              | Kịch bản áp dụng                  |
| ------------------------------------- | --------------------------------------------------------------------- | --------------------------------- |
| **A. Danh sách tra cứu nhanh Prompt** | Ôn tập khung S.C.A.F.F., các điểm chính trong kịch bản thường gặp     | Tham khảo nhanh khi viết Prompt   |
| **B. Bảng tra cứu lỗi thường gặp**    | Định vị nhanh lỗi JavaScript/CSS/Tương tác                            | Tìm giải pháp khi code báo lỗi    |
| **C. Bảng thuật ngữ**                 | Giải thích thuật ngữ liên quan đến Lập trình AI, Phát triển, Kỹ thuật | Tra cứu khi gặp từ lạ             |
| **D. Bảng so sánh công cụ**           | So sánh công cụ AI, IDE, nền tảng triển khai                          | Tham khảo khi chọn công cụ        |
| **E. Danh sách ý tưởng dự án**        | 40 ý tưởng dự án để luyện tập                                         | Tìm cảm hứng khi chưa biết làm gì |

::: tip Gợi ý sử dụng
Phụ lục là **công cụ tra cứu nhanh**, không phải giáo trình. Nếu bạn không hiểu khái niệm nào đó, kiến nghị quay lại đọc chương tương ứng trong chính văn.
:::
